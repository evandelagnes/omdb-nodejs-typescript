## How to Setup
1. Copy environment variables. Replace values to your own.
```bash
$ cp .env.example .env
```

2. Build the project with [docker-compose](https://docs.docker.com/compose/).


```bash
$ docker-compose build
```

3. Start containers.
```bash
$ docker-compose up
```
